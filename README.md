# missao3B

Projeto desenvolvido durante as aulas de programação, nas linguagens de programação HTML, CSS e JavaScript.
O presente projeto é uma história com vários caminhos a ser percorridos pelo usuário.
Feito por Luís Gustavo Lério Rodrigues, 3ºB

Nossa história será sobre:

O Apartamento 407

O prédio na Rua Alvorada foi abandonado há quase quinze anos.

Pelo menos, era isso que todos diziam.

Na tarde em que seu celular perdeu o sinal, uma notificação apareceu na tela sem conexão com a internet:

"Você esqueceu algo no apartamento 407."

O estranho é que você nunca morou naquele prédio.

Mesmo assim, a curiosidade vence o medo.

O elevador está desligado, então você sobe pelas escadas. Cada andar parece mais silencioso que o anterior. Ao chegar ao quarto andar, encontra apenas duas portas.

A primeira está identificada com o número 407. A maçaneta está quente, como se alguém tivesse acabado de entrar.

No fim do corredor existe uma porta de emergência entreaberta. Lá dentro, uma escada leva ao telhado. Uma corrente de ar frio sobe por ela, trazendo um cheiro de chuva e alguém parece chamar seu nome.

Você percebe que só tem tempo para uma escolha.
